# mpd-scripts - volume


| Script Name    | Description                                                                   |
|----------------|-------------------------------------------------------------------------------|
| [volume.py](./volume.py)           | Adjust MPD volume using mpc command-line tool.                                 |

| [mpdvoldown.py](./mpdvoldown.py)   | Decrease MPD volume using mpc command-line tool.
